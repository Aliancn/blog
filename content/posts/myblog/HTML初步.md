## Inline and Block Level Element

### 块级元素

* 在页面以块的形式展现
* 出现在新的一行
* 占全部宽度

```html
<div><h1>-<h6><p>
```

### 内联元素

* 通常在块级元素内
* 不会导致文本换行
* 只占必要的部分宽度

```html
<a><img><em><strong>
```
