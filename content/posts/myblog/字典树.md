## 创建树结构

```c++
const int kind = 26;
struct TrieNode{
    int son[kind];
    bool mark;
}trie[10001];
int root=0,num=0;
```

## 插入并判重

```c++
bool insert(char* str){
    int position=root;
    for(int i=0;str[i];i++){
        int symbol=(int)str[i]-'a';
        if(!trie[position].son[symbol]){
            trie[position].son[symbol]=++num;
        }
        position=trie[position].son[symbol];
    }
    int tmp=trie[position].mark;
    trie[position].mark=1;
    return tmp!=0;
}
```
